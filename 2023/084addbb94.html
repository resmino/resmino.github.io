Java标识符和基本数据类型

发表于 2023-08-04 分类于 学习笔记 ， Java 阅读次数： Waline： 本文字数： 365 阅读时长 ≈ 1 分钟
学习笔记-Java标识符和基本数据类型

标识符

  • 组成：字母、数字、下划线

  • 关键字：48个

  • 驼峰命名：若变量由多个单词组成，第一个单词首字母小写，其他单词首字母大写

基本数据类型

分为基本数据类型和应用数据类型。

  • 八大基本数据类型：

    • byte：1字节
    • char：2
    • short：2
    • int：4
    • long：8
    • float：4
    • double：8
    • boolean：1, true,false

  • 数据存储

    • 永久存储：MySQL、sqlserver，oracle，nosql等
    • 瞬时存储：中间数据，程序结束后空间回收

  • 常量：

    • 使用final定义
    • 命名规则：字母全大写

  • 变量：

    • 定义：根据数据类型进行定义
    • 命名：驼峰规范
    • 使用步骤
      • 声明变量：声明相应数据类型的变量，int a; (一般声明和初始化同时进行，int a = 0;)
      • 初始化：使用=赋值，为变量名指向的地址空间存入数据 a=0;
      • 使用：在代码中通过变量名操作数据 System.out.println(a); —0
    float score = 98.5f;
String course = "Java课";
String name = "张三";
char sex = '男';
System.out.println(course+"最高成绩是："+score+"\n最高分学员姓名："+name+"\n最高分学员性别："+sex);

  • 类型转换

    • 强制类型转换：使用（）进行强制类型转换，会产生精度损失
    char a = (int) 99;
System.*out*.println(a);  //   c
    • 自动类型转换：自下向上转换
    int b = 'a';
System.*out*.println(b);  //97
    • BigDecimal:用以计算过程中精度要求高，不能随意截断的数据，eg：money
